# Rahul Edu Consultancy 🎓✈️

A modern, high-converting web application for a Study Abroad Consultancy business. Designed with a focus on trust-building, aggressive lead capture, and showcasing global universities and programs with dynamic, interactive elements.

## ✨ Features

- **Modern & Interactive UI:** Premium sleek aesthetics, responsive design, and dynamic hover effects.
- **University & Country Showcases:** Beautifully displayed destination options, top universities, and realistic insights to guide aspiring students.
- **Trust Building:** Dedicated sections for student testimonials, success stories, and proven results.
- **Aggressive Lead Capture:** Optimized pathways for user engagement and lead generation.
- **SEO Optimized:** Best practices embedded to ensure maximum reach.

## 🛠️ Tech Stack

- **Framework:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Styling:** Custom CSS with modern layouts (Grid/Flexbox) and micro-animations.
- **Routing:** React Router (if applicable)

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository** (or download the source code):
   ```bash
   git clone <your-repo-url>
   ```

2. **Navigate into the project directory:**
   ```bash
   cd rahul_edu_consultancy
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

### Running the Development Server

Start the local host via Vite:

```bash
npm run dev
```
The app will typically be available at `http://localhost:5173`.

### Building for Production

To create an optimized production build:

```bash
npm run build
```
This generates a `dist` folder containing the minified files ready for deployment.
